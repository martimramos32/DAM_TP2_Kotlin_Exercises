# Exercícios de Introdução ao Kotlin

## Descrição
Este repositório contém a resolução da primeira parte do TP2 de DAM (Desenvolvimento de Aplicações Móveis). O foco principal foi a aprendizagem da sintaxe da linguagem Kotlin, manipulação de coleções, estruturas de controlo e lógica de programação orientada a objetos.

## Tecnologias Utilizadas
* **Linguagem:** Kotlin
* **IDE:** IntelliJ IDEA
* **Build System:** Gradle

## Estrutura
Os exercícios cobrem:
* Manipulação de Arrays e Listas.
* Leitura de dados de ficheiros XML (ficheiros de recursos).
* Lógica de processamento de dados para aplicações Android.

## Como Executar
1. Abrir o IntelliJ IDEA.
2. Executar as funções `main` de cada package (exer1_1, exer1_2, exer1_3 e exer1_4).
